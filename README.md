# Ragnarok Next Generation Job Class Gallery — Static Website (Activity 6 — Web Development)

This repository contains Activity 6 for a Web Development course/exercise. It is a static gallery website used to demonstrate HTML, CSS and JavaScript fundamentals and to showcase the different job classes in the game of Ragnarok Next Generation.

## What this repo contains

This repository is a simple front-end project (no build step). It's intended for learning, small demos, and quickly previewing an HTML/CSS/JS static page featuring a responsive card-based gallery layout.

### File structure

Root files

- `index.html` — Main site entry (single page).
- `README.md` — This file.

Assets

- `assets/css/style.css` — Main CSS stylesheet with custom Ragnarok-themed styling.
- `assets/img/` — Images used by the site (logos and icons).
- `assets/script/data.js` — Job class data and content.

Directory tree (visual):

```
index.html
README.md
assets/
  css/
    style.css
  img/
    ROX_logo.png
    Javillo_logo.svg
  script/
    data.js
```

## Features

- **Responsive Design**: Fixed-width card layout that adapts from 1 column (mobile) to 2 columns (desktop)
- **Custom Theming**: Medieval Ragnarok-inspired brown color palette with custom fonts (Cinzel & MedievalSharp)
- **Scroll Animations**: Fade-in-up animations as cards enter the viewport
- **Blurred Background**: Card background with gradient overlay for improved readability
- **Bootstrap 5.3.2**: Utilizes Bootstrap grid system and utilities
- **Dynamic Content**: Job class cards generated dynamically from JavaScript data

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Animations)
- JavaScript (ES6, IntersectionObserver API)
- Bootstrap 5.3.2
- Google Fonts
